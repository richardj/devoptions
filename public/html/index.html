Setting up development environment

Have your prerequisites ready, after following the install instructions for all of the prerequisites

When NodeJS is installed you have access to NPM as well, which is quite useful to install all your modules

Setting up project

Start with running npm init. This will initialize your package.json file which holds all the information for the packages included in your projects.

After filling out all the questions from the npm init command you can start getting the packages you want.

Some modules need to be installed globally, do that by adding the -g flag.
Use npm install PACKAGENAME to install a package, if you want to save that to your package.json file (most of the times you want that) just add the --save flag. If the package is only needed during development, you add the --save-dev flag.

Example

Let's take an example to get started without going into to much detail, we want to use AngularJS in our project

  • npm install angular --save
  • ... installs angularJS to the node_modules/ folder and saves that instruction to your package.json file
  • Add the following in your HTML file
  • <script src="/node_modules/angular/angular.js"></script>
  • AngularJS should be included now and available
